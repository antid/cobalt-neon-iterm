# My Project

This project is a website designed to showcase my portfolio and provide information about me and my work.

## Getting Started

To get started, you should visit the homepage of my website to learn more about me. You can also view my portfolio of work, read my blog posts, and contact me for any inquiries.

## Features

My website features a portfolio of my work, a blog with updates about my life and work, and a contact page to reach out to me.

## Technology Used

This website is built with HTML, CSS, and JavaScript. I used Bootstrap 4 for styling, and Google Fonts for the font library.
